# CLD

CLD is a secure peer-to-peer terminal messaging system built in Rust.

It enables direct encrypted communication between trusted devices over a private network without relying on centralized messaging servers.

CLD focuses on:

* peer-to-peer networking
* terminal-first UX
* modern cryptography
* local-first architecture
* systems programming

Built with:

* Rust
* Tokio
* Ratatui
* SQLite
* X25519
* HKDF
* ChaCha20Poly1305
